# Param Description  
パラメータ調整の行い方・変数の説明  
  
# Content  
- mimi_motor.yaml  
motor_setup.launchで使用するyamlファイル  
各モータの設定を記述  
使用するモータの数に変更があったり、速度を変更したい場合はこのファイルの中身を変更してください。  
- mimi_specification.yaml  
各種ノードで使用するyamlファイル  
ハードウェアの長さやアームの規格などmimiの仕様を記述
